# SafeSphere – Women Safety Web App

SafeSphere is a single-page, offline-friendly web application designed to enhance women’s safety in urban areas like Kolkata.

## Features
- SOS emergency call
- SOS SMS with location
- West Bengal women helplines
- Offline incident reporting
- Location sharing
- Safety tips

## Technology Used
- HTML
- CSS
- JavaScript

## Uniqueness
- Works offline
- No login required
- Single-page architecture
- Region-specific focus
